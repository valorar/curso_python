# curso_python

Ejercicios propuestos para un curso de Python.

## Estructura

- `propuestos/` — Ejercicios propuestos organizados por nivel para que los alumnos practiquen de forma activa los conceptos aprendidos.

## Metodología

Cada nivel incluye:
- Retos resueltos con vídeo explicativo
- **Retos propuestos** (ejercicios análogos para resolver de forma autónoma)
- Mini-proyectos que integran varios conceptos
- Ejercicios de código roto para depuración
- Preguntas de reflexión
